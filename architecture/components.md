# Platform Components

## PostgreSQL
Primary transactional database with streaming replication.

## WAL Archive
Stores WAL segments for disaster recovery.

## TimescaleDB
Time-series analytics database.

## Redis
Caching layer for application performance.

## Prometheus
Central metrics collection system.

## Grafana
Visualization dashboards.

## Alertmanager
Alert routing and notification management.

## Fluent Bit
Log collection and transformation agent.

## ClickHouse
Column-oriented database for log analytics.
